# Hotel Reservation System

## Overview

A Java-based Hotel Reservation System that allows users to view rooms, make reservations, cancel bookings, and view booking details.

## Features

* View available rooms
* Room categorization (Standard, Deluxe, Suite)
* Book hotel rooms
* Cancel reservations
* Payment simulation
* View booking details
* File handling for storing bookings

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* File Handling
* ArrayList
* Scanner Class

## Project Structure

* Room.java
* Booking.java
* HotelReservationSystem.java
* bookings.txt

## How to Run

Compile:

javac Room.java
javac Booking.java
javac HotelReservationSystem.java

Run:

java HotelReservationSystem
